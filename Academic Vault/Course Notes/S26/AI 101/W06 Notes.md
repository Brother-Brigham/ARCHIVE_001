## May 26 Class
**Presentations Continued**
- ReAct AI Prompting
	- ReAct stands for Reason + Acting
		- Aimed to minimize hallucinations
	- Steps:
		- Reasons about what must be done
		- Acts (such as selecting an available tool)
		- Observes its output
		- Reacts and continues refining output
	- Better reasoning for complex problems
	- Reduces hallucinations
	- Works well with tools
	- Improves transparency
	- Strong for agents and autonomous workflows
	- Can still hallucinate
	- Should be fact checked
- Graph Prompting
	- Incredibly useful for developing true logical connections, especially useful for e-commerce systems
	- Follows a map instead of text
	- High accuracy
	- Follows physical paths
	- Handles millions of links
	- Provides a massive reliability boost for complex prompts
**Human-in-the-Loop (HITL)**
- Tokens
	- One token is four characters
	- Context windows are limited to token amounts
		- Many frontier context windows are able to handle gargantuan numbers of tokens
	- Conversations use limited amounts of the context window, but coding applications are liable to use more.
- HITL has moved from *muscle* to *brain* to *teacher*, is currently at *manager*, and in the future may become *governor* and *expert collaborator*.
- Is there value in the process, or is the end product all that matters?
	- The process is perhaps the most meaningful, or at least the most meaning-infusing part of the experience.
