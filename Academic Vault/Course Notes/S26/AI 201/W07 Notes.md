## June 4 Class
**Task Markdown File**
- Consists of the following parts:
	- Role
		- What role is the AI playing?
		- What attitude should an exceptional performer have?
	- Task
		- Concise and precise definition of the task
		- Name every task and sub-task
	- Steps
		- Describe how the task is to be done
	- Analysis
		- Provide the human agent manager with all the details required to make a well-informed decision about a solution
	- Examples
		- One or more examples that begin with the initial data and conclude with the final solution
